🎓 Student Management System

A simple Python console-based application to manage student records using file storage.

📌 Features

- Add Student
- View Students
- Search Student
- Delete Student
- Data stored in a text file

🛠 Technologies Used

- Python
- Lists
- Loops
- File Handling

▶️ How to Run

python student_management.py
