🎓 Student Management System (CLI + GUI)

A beginner-friendly Python project that demonstrates how to build a Student Management System using both Command Line Interface (CLI) and Graphical User Interface (GUI).

---

📌 Project Description

This project allows users to manage student records efficiently.
It includes two versions:

- 💻 CLI Version (Terminal based)
- 🖥 GUI Version (Tkinter based)

The system supports adding, viewing, searching, and deleting student records.
All data is stored in a file ("students.txt") for persistence.

---

⚙️ Features

🔹 Common Features

- Add Student
- View Students
- Search Student
- Delete Student
- Data stored in file ("students.txt")
- Duplicate prevention
- Case-insensitive search

🔹 CLI Features

- Menu-driven interface
- Simple and lightweight
- Easy to run in terminal

🔹 GUI Features

- Interactive buttons
- Listbox display
- Scrollbar support
- Popup messages (success/error)
- User-friendly interface

---

🛠 Technologies Used
